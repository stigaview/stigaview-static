# STIG A View Static

## Input
Below is an outline how the `products` directory is layout out.
```
products
    rhel7
        v1r1.xml
        v1r2.xml
    rhel8
        v3r1.xml
    ubuntu2004
```

## Usage
You will need Python 3.11+.

```
$ python -m stigaview_static -o out products
```
